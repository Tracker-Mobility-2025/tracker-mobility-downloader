# Coloca tu archivo APK aquí

Esta carpeta está destinada para almacenar el archivo APK de la aplicación Tracker Mobility.

## Instrucciones:

1. **Coloca tu archivo APK** en esta carpeta
2. **Renómbralo** a `tracker-mobility-worker.apk`
3. O si prefieres usar otro nombre, edita la variable `CONFIG.APK_FILE` en `src/assets/js/app.js`

## Ejemplo de estructura:

```
downloads/
└── tracker-mobility-worker.apk    # Tu archivo APK aquí
```

## Notas importantes:

- El archivo APK debe ser válido y funcional
- Asegúrate de que el archivo no esté corrupto
- Verifica que sea la versión correcta para producción
- El tamaño del archivo debe coincidir con la información mostrada en el sitio web

---

**Recuerda:** Este archivo será descargado directamente por los trabajadores de campo, asegúrate de que sea la versión más reciente y estable.
